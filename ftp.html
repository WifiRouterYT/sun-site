FTP (WIP!)

I host a small collection of files/media I archived, as well as mirrors of Sun Microsystems related FTP sites, and a few other long shut down or cool FTP sites just for the heck of it. The FTP server can be accessed in two ways:

  • via HTTP at http://ftp.wifirouter.dev/
  • via your favorite FTP client (Open)
    • Host: ftp.wifirouter.dev
    • Port: 21
    • Directory: /pub
    • Login: anonymous

Mirrors

You can find information about an FTP mirror by looking for the following files in the mirrors folder:

  • MIRRORINFO - Contains general information about the mirror (source, original site still active, when it was first added, etc.)
  • MIRRORSTATUS - Contains information about the mirror status (last updated, ommited/failed files, etc.)

General info and summary of all of the mirrors can be found in the README file in the root of the mirrors directory. For the sake of saving storage space and bandwidth, most mirrors will be partial mirrors. You can request content from a partial mirror to be added if you wish.

I did not manually review all the little nooks and crannies of site mirrors, so tread carefully. I try to mirror from reputable and safe sources, but things can slip in without my (or the original site owners) knowledge.

Compilations

In the FTP server, you'll find folders with a bunch of similar files grouped together, such as Solaris packages, to help find common files from mirrors and other sources easier. These files are not symlinked for the sake of compatibility with Archie and older clients, so it's possible that if something is updated it may be missing. If that's the case, please let me know and I'll fix it.
